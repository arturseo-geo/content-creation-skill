High-quality content creation skill for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) — blogs, landing pages, email sequences, social media, video scripts, newsletters, case studies, whitepapers, content calendars, editorial briefs, and cross-platform repurposing.

## Who This Is For

- **Content marketers** who want Claude to produce publication-ready drafts, not generic AI text
- **Solopreneurs** handling their own content across multiple platforms
- **Agency teams** who need consistent tone of voice across deliverables
- **Anyone creating content** who wants platform-specific formatting done right the first time

## What Makes This Different

Generic AI writing tools produce generic output. This skill knows platform constraints, formatting rules, and repurposing workflows:

- ✅ **12+ content types** — blog posts, landing pages, emails, LinkedIn, X/Twitter, Instagram, YouTube scripts, newsletters, case studies, whitepapers, product descriptions, video scripts
- ✅ **Platform-specific rules** — character limits, formatting quirks, optimal posting times
- ✅ **Ready-to-use templates** — structured templates for every common format
- ✅ **Repurposing workflows** — blog → LinkedIn thread → email → social carousel
- ✅ **SEO writing integration** — on-page best practices built into content templates
- ✅ **Tone calibration** — adapts to brand voice, not just "professional" or "casual"

## Install

```bash
# Clone
git clone https://github.com/arturseo-geo/content-creation-skill.git ~/.claude/skills/content-creation

# Or install all 12 skills at once
git clone https://github.com/arturseo-geo/claude-code-skills.git
cp -r claude-code-skills/skills/content-creation ~/.claude/skills/
```
